# OZE Lublin

Lokalny portal o odnawialnych źródłach energii w Lublinie i na Lubelszczyźnie.

## Stack
- Astro 4.x (static output)
- Vanilla CSS (Warm Terracotta — local community feel theme)
- Cloudflare Pages deployment

## Uruchomienie
```bash
npm install
npm run dev
```

## Deploy
Cloudflare Pages: build `npm run build`, publish dir `dist`
